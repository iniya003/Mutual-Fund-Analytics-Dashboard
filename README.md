 **Mutual Fund Analytics Dashboard**

**Project Overview**
This project is an end-to-end Mutual Fund Analytics Dashboard developed using Python, SQL, Power BI, and Jupyter Notebook. It analyzes mutual fund performance, investor behavior, portfolio risk, and market trends to provide actionable investment insights through interactive dashboards and advanced analytics.

---

**Features**
- Data Cleaning and Preprocessing
- Mutual Fund Performance Analysis
- NAV Trend Analysis
- Risk Analysis (VaR, CVaR, Sharpe Ratio)
- Investor Cohort Analysis
- SIP Continuity Tracking
- Fund Recommendation Engine
- Sector Concentration Analysis (HHI)
- Interactive Power BI Dashboard
- Business Insights and KPI Reporting


 Technologies Used
- Python
- SQL
- Power BI
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook


Project Workflow
1. Data Collection
2. Data Cleaning & Validation
3. Exploratory Data Analysis (EDA)
4. Risk Metrics Calculation
5. Investor Behavior Analysis
6. Fund Recommendation System
7. Power BI Dashboard Development
8. Business Insights & Reporting



 Dashboard Highlights
- Total Assets Under Management (AUM)
- SIP Inflow Analysis
- Mutual Fund Performance Trends
- Investor Growth Analysis
- Risk & Return Comparison
- Top Performing AMCs
- Portfolio Diversification Analysis
- Interactive Filters and Visualizations

 Key Outcomes
- Improved understanding of mutual fund performance.
- Identified investment risks using financial metrics.
- Analyzed investor behavior and SIP continuity.
- Generated fund recommendations based on risk appetite.
- Built an interactive dashboard for business decision-making.
